# Plan: 보고서 UI 고도화 및 품질 개선 (UI Refinement & Bug Fix)

## 1. 개요 (Overview)
- **목표:** 기존 Building Report Pro의 보고서 품질을 실무급으로 격상시키고, 발견된 층 표시 오류를 해결하여 신뢰성을 확보함.
- **핵심 개선 사항:** 고대비 블랙 & 화이트 디자인 적용, 층 표시 로직 정상화, 용어 한글화 및 단순화.

## 2. 세부 과제
1. **[Check/Act] 층 표시 오류 해결**: `flrGbCd`를 활용하여 지하(B), 옥탑(R) 등을 정확히 표시하도록 `parseFloor` 로직 보완.
2. **[Design/Do] B&W 프리미엄 디자인**: 기존의 컬러풀한 테마에서 벗어나 전문 비즈니스 문서 느낌의 모노톤 디자인으로 전면 개편.
3. **[Do] 용어 정제**: `CAP RATE`, `ROE` 등 전문 용어를 `예상 수익률`, `내 돈 대비 수익률` 등 쉬운 한글로 교체.

## 3. 일정 (Roadmap)
- **P1:** 층 표시 오류 분석 및 해결 (완료)
- **P2:** B&W 디자인 시스템 적용 (완료)
- **P3:** 유저 피드백 기반 용어 정제 (완료)
- **P4:** PDCA 문서 체계 동기화 (진행 중)
