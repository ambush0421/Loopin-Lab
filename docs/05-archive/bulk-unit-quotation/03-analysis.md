# Gap Analysis: Bulk Unit Quotation (UX/Value-up)

## 📊 Match Rate: 85%

### 1. 설계 vs 구현 비교
- **UI/UX (90%):** 3가지 타입의 전문 견적서 레이아웃이 완벽하게 구현됨. 분할 화면(Split View)으로 실시간 피드백 제공.
- **데이터 로직 (95%):** 일괄 수정 및 개별 수정(Override) 로직이 견고하게 작동함. NOC 등 실무 지표 자동 계산 포함.
- **가독성/용어 (70%):** 전문 용어(NOC, Yield, Cap Rate 등)가 다수 포함되어 일반인이나 초보 실무자가 이해하기 어려움. 레이아웃이 빽빽하여 시각적 피로도 존재.

### 2. 발견된 문제점 (Issues)
- **용어 장벽:** 'NOC', 'Cap Rate', 'Investment Memorandum' 등 영어 위주의 전문 용어가 비전문가에게 진입장벽이 됨.
- **시각적 복잡도:** 왼쪽 패널의 슬라이더와 입력창이 밀집되어 있어 중요한 '결론' 보다는 '수치 조정'에 에너지가 많이 쓰임.
- **Value-up 부족:** 데이터는 정확하나, "이 건물이 왜 좋은가"에 대한 시각적 강조(Highlight)가 부족함.

### 3. 개선 계획 (PDCA Iterate)
- **용어 순화:** 전문 용어를 직관적인 한글 용어로 병기 또는 대체 (예: NOC -> 평당 실제 점유비).
- **레이아웃 밸류업:** 화이트스페이스를 더 확보하고, 핵심 숫자의 폰트 웨이트와 크기를 조정하여 '시선이 머무는 곳'을 명확히 함.
- **안정성 강화:** 수기 입력 시 발생할 수 있는 0나누기(NaN) 에러 방지 로직 강화.
