# Design: 대량 호실 견적 및 건물 제원 분석 시스템 (Bulk Unit Quotation & Spec)

## 1. UI/UX 설계 (밸류업 버전)

### 1.1 건물 제원 인포그래픽 (Building Spec Section)
대시보드 상단 또는 지표 카드 영역에 다음과 같이 배치합니다.

```
┌─────────────────────────────────────────────────────────────┐
│ [건물 규모 및 컨디션 요약]                                     │
├─────────────────────────────────────────────────────────────┤
│ ┌────────────┐ ┌────────────┐ ┌────────────┐ ┌────────────┐ │
│ │ 층수: B5/20F│ │ 연면적: 5만㎡│ │ 사용승인: 2023│ │ 주차: 500대│ │
│ │ (지하/지상) │ │ (약 1.5만평) │ │ (신축 3년차)  │ │ (호실당 1.2)│ │
│ └────────────┘ └────────────┘ └────────────┘ └────────────┘ │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 핵심 컴포넌트 추가
- **SpecDashboard:** 
  - `ugndFlrCnt`, `grndFlrCnt` 데이터를 활용한 층수 요약.
  - `totArea`를 평 단위로 환산 표시.
  - `useAprvDay`를 날짜 형식(`YYYY.MM.DD`)으로 가공하고 현재 날짜 기준 연차 계산 표시.

## 2. API 및 데이터 가공

### 2.1 데이터 변환 로직
- **연차 계산:** `new Date().getFullYear() - parseInt(useAprvDay.substring(0, 4))`
- **평수 환산:** `totArea * 0.3025`
- **층수 포맷:** `B${ugndFlrCnt} / ${grndFlrCnt}F`
