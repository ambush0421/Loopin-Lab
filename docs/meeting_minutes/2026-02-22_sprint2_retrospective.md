# [2026-02-22] 제2차 B2B 리포트 품질 검증 QA 및 회고록 (Meeting Minutes)

---

## 1. [Plan] 목적 및 달성 내역

- **목적:** 1차 회의에서 도출된 가상 유저 피드백(김대표)과 B2B 백로그를 바탕으로, **[동적 금리 시뮬레이터 UI]**를 추가하고 고객사 브랜딩을 위한 **[보고서 커스텀 로고 삽입]** 기능을 구현 완료하는 것.
- **달성률:** 100% 완료 (슬라이더 UI 연동 100%, 파일 Base64 변환 로고 삽입 100%)

## 2. [Do] 수행된 코드 수정 내역

- **`/engineer` & `/designer` 협업:**
  - `QuotationModal.tsx` 내 `assumptions` 상태에 금리(`interestRate`), 실입주 LTV(`loanLtvUse`), 투자 LTV(`loanLtvInvest`)를 즉각적으로 변경할 수 있는 Range Input(슬라이더) UI 개발 및 연동 완료.
  - 모달 상단 'PDF 리포트' 버튼 옆에 커스텀 로고 업로드를 위한 `<input type="file" />` 히든 요소 및 버튼 구현. 파일 업로드 시 `FileReader`를 통한 Base64 렌더링 도입.
  - `PremiumReportTemplate.tsx`를 통해 `ReportCoverPage.tsx`까지 업로드된 `customLogo` 파라미터를 Prop-Drilling 방식으로 안전하게 전달하여 표지 인쇄 시 즉시 반영되도록 구현.

## 3. [Check] 가상 사용자의 날카로운 피드백과 QA 이슈 내역

- **가상 사용자 페르소나 (최이사, 40대, 자산운용사 본부장):**
  - *"슬라이더로 금리를 조절하니 밑에 월 지출 차트랑 자기자본수익률(ROI) 숫자가 바로바로 뚝뚝 떨어지는 게 보이니 직관적이네요. 다만 현재 투자 LTV를 0%부터 90%까지 당길 수 있는데, 현실적으로 상가를 90% 대출받는 경우는 불법이거나 제2금융권을 껴야 해서 흔치 않습니다. 슬라이더 상한선을 조금 방어적으로 두거나 경고(Tooltip)가 뜨면 좋겠어요."*
- **가상 사용자 페르소나 (박대표, 50대, 개인 공인중개사):**
  - *"우리 중개사무소 로고를 올리니 보고서가 확 사네요. 그런데 이미지 비율이 깨지진 않는지 걱정입니다. 가로로 좀 긴 로고를 올렸을 때 문제없었으면 해요."*
- **`/qa` 에이전트 발견 이슈:**
  - `img` 태그에 대한 Vercel/Next.js Lint 룰 경고(Warning) 발견
    (`Using <img> could result in slower LCP... Consider using <Image />`).
    현재 PDF Export 라이브러리 연동 및 로컬 업로드 특성상 표준 `<img>`를 사용하는 것이 안전하므로 Lint Disable 처리를 하거나 의도된 예외로 간주함 (코드 정상 동작 확인됨).
  - 디자인 적용 시 슬라이더(range) 모양이 브라우저(Chrome/Safari)마다 조금 다르게 렌더링되며, 색상이 일부 어긋남을 발견.

## 4. [Act] 가상 사용자 피드백을 반영한 1차 수정/대응 결과

- **로고 비율 최적화:** `/designer`가 `ReportCoverPage.tsx`의 이미지 렌더링 클래스에 `object-contain` 및 `max-h-16` 제한 유틸리티를 적용하여 박대표(개인 공인중개사) 님의 가로형/세로형 로고 찌그러짐 이슈 사전 차단.
- **슬라이더 크로스브라우징:** `/engineer`가 Webkit Appearance-none 속성과 전용 Accent 렌더링 클래스를 적용해 통일된 UI 확보.

## 5. [Value-up 개선 포인트]

- **대출 한도 규제 로직(DSR / RTI 방어):**
  최이사(자산운용사 본부장)의 지적에 따라 LTV나 금리를 무한정 올리는 것이 불가능하게 만들거나, 경고 문구를 띄우는 **Compliance Check(규제 심사)** 기능의 도입 구상 필요.
- **권리금 & 부가가치세(VAT) 모듈 연동:**
  단순 임대차 현금 흐름을 넘어 부가세 환급 시나리오와 권리금 감가상각을 포함한 심층 세무 분석 추가 필요성 대두 (기존 이대리의 피드백과 연결).
